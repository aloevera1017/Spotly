Spotly combines AI-powered predictions with real-time community insights to help you find available parking spots instantly. Never circle the block again!

---

## ✨ What is Spotly?

Spotly is an intelligent parking assistant that:
- 🤖 **Predicts** parking availability using machine learning
- 📍 **Tracks** 100,000+ individual parking spaces across NYC
- 🔍 **Searches** by address with customizable radius
- 🌐 **Crowdsources** real-time conditions from the community
- 📸 **Validates** predictions with user-submitted photos
- 📊 **Analyzes** patterns to find you the best spots

---

## 🎯 Key Features

### 🏠 Main Dashboard
- **Individual Space Tracking**: See availability for every single parking spot
- **Smart Address Search**: Find parking near any NYC location
- **Radius Filtering**: Search within 0.5km to 5km of your destination
- **Time-Based Predictions**: Check availability for any future date/time
- **Interactive Map**: Visual, color-coded availability indicators
- **Real-Time Filtering**: By borough, facility type, and availability
- **CSV Export**: Download predictions for offline use

### 🌐 Community Insights
- **Contribute**: Share real parking conditions with photos
- **View**: Browse community-submitted insights
- **Engage**: Like and mark insights as helpful
- **Filter**: Find relevant insights by location and time
- **Statistics**: See platform-wide parking trends

## 🚀 Quick Start

### Access Spotly
```
Open your browser to: http://localhost:8503
```

### Find Parking in 3 Steps

1. **Search Your Destination**
   ```
   Enter: "Times Square" or "123 Main St"
   Set radius: 1.0 km
   Click: 🔍 Search
   ```

2. **Set Your Time**
   ```
   Select when you need parking
   Or use quick presets (Rush Hour, Lunch Time, etc.)
   Click: 🚀 Predict Availability
   ```

3. **Pick Your Spot**
   ```
   See green markers = available
   Click markers for details
   View individual space numbers
   Get directions!
   ```

## 🧠 How Spotly Works

### AI Prediction Engine

Spotly uses advanced algorithms to predict parking occupancy based on:

**Time Patterns**
- Hour of day (rush hour vs off-peak)
- Day of week (weekday vs weekend)
- Business hours vs off-hours
- Seasonal variations

**Location Intelligence**
- Borough-specific patterns
- Commercial vs residential areas
- Meter type (passenger vs commercial)
- Traffic flow analysis

**Random Realism**
- Natural fluctuations
- Unexpected variations
- Event-based adjustments

### Community Validation

Real drivers share:
- Current parking conditions
- Photo evidence
- Traffic levels
- Special circumstances

This crowdsourced data helps validate and improve our AI predictions!

## 💡 Tips & Tricks

**Get the Best Results**
1. Search 10-15 minutes before you need parking
2. Set radius to 1-2km for good options
3. Check multiple boroughs if flexible
4. Use quick presets for common times
5. Sort by distance when in a hurry

**Contribute Quality Insights**
1. Take clear photos of parking areas
2. Include helpful context in notes
3. Update traffic level accurately
